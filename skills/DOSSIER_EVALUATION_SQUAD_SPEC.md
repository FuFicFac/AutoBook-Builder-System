# Dossier Evaluation Squad Spec

## Purpose
Pre-draft readiness auditing across five specialist perspectives.

## Personas
- ATLAS: structural causality and beat integrity
- PSYCH: character logic and arc believability
- NOVA: world specificity and rule coherence
- TEMPO: pacing and escalation rhythm
- NICHE: audience/genre promise alignment

## Input
- `DOSSIER.md` (required)
- `VOICE_PROFILE.md` (optional)

## Full Output Order (Strict)
1. ATLAS
2. PSYCH
3. NOVA
4. TEMPO
5. NICHE
6. CONSENSUS SUMMARY
7. FOUNDATION VERDICT
8. NEXT STEPS

## Scoring Bands
- 1–4: STOP
- 5–6: CAUTION
- 7–8: SOLID
- 9–10: GREENLIGHT

## Arbitration Priority
Structure > Character > World > Pacing > Market

## Repair Loop Trigger
If any persona <= 4:
- block drafting
- emit fix-first list
- ask targeted repair questions
- re-evaluate only impacted personas
