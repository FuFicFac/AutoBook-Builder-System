# Artifact Schemas

## DOSSIER.md (Minimum Sections)
- Title and logline
- Genre, audience, tone
- Promise and hook
- Theme and central dilemma
- Protagonist + arc direction
- Antagonistic force
- Supporting cast
- World rules/constraints
- Stakes and escalation ladder
- Act/beat spine
- Set pieces
- Must-have/must-avoid
- Ending profile

## VOICE_PROFILE.md
- Voice descriptors
- Narrative distance, POV, tense
- Prose texture and rhythm
- Dialogue style
- Emotional temperature
- Pacing profile
- Style boundaries

## BBB_READINESS_REPORT.md
- Completeness score
- Critical gaps
- Assumptions
- Handoff state
- Next required question (if not ready)

## Build Log Schema
- Event ID
- Chapter reference
- Time marker
- Event summary
- Impacted entities
- Follow-up/payoff tag

## Continuity Bible Schema
- Character states
- Location states
- Object states
- Timeline
- Setups/payoffs
- Open threads

## Session Memory Schema
- Current drafting scope
- Active threads
- Emotional pacing snapshot
- Immediate next steps
- Assumptions list
